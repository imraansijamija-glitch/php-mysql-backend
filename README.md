# PHP MySQL Backend CRUD

## Student
Imraan Sijamija

## Predmet
Softver inžinjering
## Opis projekta
Web aplikacija za upravljanje korisnicima koristeći PHP i MySQL bazu podataka.

Aplikacija omogućava:
- Create (dodavanje korisnika)
- Read (prikaz korisnika)
- Update (izmjena podataka)
- Delete (brisanje korisnika)

- PHP
- MySQL
